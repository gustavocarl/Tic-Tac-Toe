## Introduction

Welcome to the Tic-Tac-Toe Console Application! This simple yet fun game allows two players to compete in the classic game of Tic-Tac-Toe right in your console.

## How to Play

1. Start the game by running the program.
2. Follow the on-screen prompts to enter the players' names.
3. Take turns choosing a position on the board by entering a number from 1 to 9.
4. The first player to get three of their symbols in a row (horizontally, vertically, or diagonally) wins the game.
5. If no one wins after all positions are filled, the game ends in a draw.

## Features

- User-friendly console interface.
- Two-player mode.
- Keeps track of game statistics.
- Option to reset game statistics.

## Installation

1. Clone this repository to your local machine.
2. Build the solution using Visual Studio or the C# compiler.
3. Run the generated executable file.

## Usage

To start the game, run the executable file. Follow the on-screen instructions to play Tic-Tac-Toe with a friend.

## Statistics

You can view game statistics by choosing the "Ver estatísticas de vitórias" option in the menu. The program will display a list of player names who have won games.

## Resetting Statistics

If you wish to reset the game statistics, choose the "Zerar estatísticas de vitórias" option in the menu. This will delete all saved statistics.

## Directory Creation

he application automatically creates a directory at **C:\Statistics** to store game statistics. If the directory does not exist, it will be created.

## Delete Statistics Directory

To delete the statistics directory and all its contents, choose the "Zerar estatísticas de vitórias" option in the menu.
